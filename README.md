# StyleJS

The Front-End Coding Style Service.

## Installation

```
git clone https://github.com/codepier/stylejs

// Vendors
composer install
yarn install

// ENV Setup
cp .env.example .env

// APP Setup
php artisan key:generate
php artisan migrate
php artisan db:seed
```
